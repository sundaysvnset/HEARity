# HEARity: Speech-to-Text Summarization Berbasis Generative AI untuk Meningkatkan Aksesibilitas bagi Penyandang Gangguan Pendengaran

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
